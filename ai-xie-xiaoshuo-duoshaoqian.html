一句话结论：AI写小说的成本 = 平台费 + API Token费。用按量付费(BYOK)模式写一部完整长篇，Token费通常十几到几十元，加上少量月费即可。真正费钱的不是AI，而是"套餐制里用不完的额度"。想省钱就选按量付费、自选模型。

一、成本到底由什么构成？

  • 平台费：写作平台的月订阅，通常几元到几十元。
  • API Token费：真正的大头，按你调用模型的字数(Token)计费，模型越贵单价越高。

所以同样写一部书，选便宜模型 vs 顶级模型，成本可能差好几倍。

二、两种收费模式对比

模式怎么收常见坑适合
套餐制(买字数包)一次买N万字用不完就浪费，模型质量不透明，可能被赚差价用量稳定的重度用户
按量付费/BYOK平台少量月费+按实际Token付需自己去服务商申请Key(3分钟)用量不稳定/想省钱

四、常见问题（FAQ）

AI写一部长篇小说大概多少钱？
按量付费模式下，中等模型的Token费约十几到几十元，加少量平台月费，比套餐制省很多。
套餐制和按量付费哪个划算？
用量不稳定选按量付费。套餐常见"买30万用5万"的浪费，按量只为实际用量买单。

你自己去服务商拿密钥填入平台，平台只收少量使用费、不赚差价，成本更透明。
